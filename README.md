# Faceit stats saver



This is a FaceIt player stats collector I made for Counter-Strike 2.





# Version



This is the MVP of this application. 



# Application features



Currently, the application can collect game statistics for a specific date and calculate average kills, hs%, and W/L. Currently the number of simultaneously processed matches is 100



# How to use



Now you need to enter your FaceIt nickname and select match dates. Real-time statistics collection and saving are not yet implemented.
